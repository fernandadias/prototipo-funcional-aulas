Como usar

Cada slide é um <section>

  • data-type define o layout: cover, bullets, code, demo, compare, quote, agenda, image, video, chart, links, canvas
  • data-reveal="true" faz bullets aparecerem um a um com Space
  • data-notes guarda anotações que aparecem na tecla N
  • O chrome (toolbar, controles, overlays) é injetado pelo deck.js automaticamente
