## Módulo 1 — Princípios e framework

**Descrição:** Os 4 níveis de protótipo. O framework de decisão das 5 perguntas. Os 3 erros clássicos de quem prototipa com IA.

**Aulas:**

1. Por que protótipo estático não basta mais
2. Os 4 níveis de protótipo
3. O framework de decisão
4. Os 3 erros que designers cometem ao prototipar com IA

## Módulo 2 — Componentes funcionais

**Descrição:** Tabela complexa, drag-and-drop, motion com feedback. Tudo no Figma Make. Do zero até um componente funcional.

**Aulas:**

1. Panorama das ferramentas opinativas
2. Componente 1: tabela com filtros
3. Componente 2: drag and drop
4. Componente 3: feedback dinâmico

## Módulo 3 — Fluxos completos com agentes de código

**Descrição:** Agora vamos usar agentes de código e criar jornadas inteiras. Estado em três níveis: memória, sessão e banco. Quando subir cada degrau sem virar overengineering.

**Aulas:**

1. Setup do ambiente
2. O repositório base e a primeira tela
3. Estado na memória: useState e useReducer
4. Estado na sessão: persistência sem backend
5. Banco de dados real com Supabase

**Descrição:** Faker.js com dados BR. Admin pra criar perfis de teste. Overlay que troca entre vazio, cheio, loading e erro com um clique.

**Aulas:**

1. Por que dado mock importa
2. Faker.js: dados realistas em escala
3. Mock Service Worker (MSW): simulando uma API
4. O admin de perfis de teste
